## Naming convention

Please follow the existing json file format and increate the `No.json` and make a pull request.

**File path**: `${level}/${type}/${no}.json`

- `level`: N1, N2, N3
- `type`
  - grammar: 文法
  - volcabulary:　語彙
  - word:　文字
